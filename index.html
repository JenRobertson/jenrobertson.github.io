Peckin' Pixels

Play GameRepo

Peckin' Pixels is build using vanilla javascript and HTML canvas. No libraries or game engines were used, as I really enjoy making things from scratch.

It is a drag and drop based game where you play as a chicken farmer. You must look after your chickens, raise their chicks or sell their eggs for profit. The chickens in the game make their own 'decisions'.

All art and coding is done by me!

Soft Swap

Play GameRepo

Soft Swap is build using vanilla javascript and HTML canvas. No libraries or game engines were used, as I really enjoy making things from scratch.

Use the mouse to select two toys to swap. If 3 or more of the same toy are touching, they dissapear. Look out for the powerups too!

All art and coding is done by me, this time using Adobe Illustrator!
